== Description ==
Arastoo GMap Extended 

This plugin can help you embed google map with multiple markers on your web pages.

Use the shortcode [arastoo_gmap] to embed map in your pages.

Language used: English 

=== Instruction ===
1. Upload Arastoo Custom Post Type to your blog.
2. Install
3. Activate
4. Visit plugin Settings > Arastoo GMap
5. Add google map API key to remove "For development purposes"overlay on the map.
6. Add latitude and longitude of the center of the map.
7. Add you marker/s using the format below
	**Example**
	["Edison, NJ", 40.494921,-74.3703325 , 8],
	["Chino, CA #1", 33.9976897,-117.7142156 , 2]
8. Add this shortcode [arastoo_gmap] to your page.
9. Once shortcode was added, add map height to your style.css 
   	**Example**
	#map { height: 470px; }

== Frequently Asked Questions ==
= 1. How many markers can I add? =
As many as you want.
= 2. Is it required to add google map api key? =
Google map will work even without API key, but there will be an overlay "For development purposes" on the map.  
In able to remove this, you need to add API key.
